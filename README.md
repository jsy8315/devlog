  # Walker's Devlog

  > 프레임워크 없이 Vanilla JavaScript로 직접 구현한 개인 포트폴리오                                       
  
  ## 프로젝트 목적                                                                                                      
                                                                                                                         
  - 프레임워크 추상화 없이 **DOM API · 이벤트 처리 · 반응형 레이아웃**을 직접 활용하며 웹 기초 원리 이해                   
  - 프론트엔드 전 과정(설계 → 개발 → 배포 → 운영) 직접 경험
                                                                                                                           
  ## 기술 스택                                                                                                         
                                                                                                                           
  - **언어·마크업**: HTML5, CSS3, Vanilla JavaScript (ES6+)                                                                
  - **배포**: AWS [S3·CloudFront 등]
  - **외부 라이브러리·프레임워크**: 사용 안 함 (학습 목적으로 의도적 제외)                                                 
                                                                                                                           
  ## 페이지 구성                                                                                                        
                                                                                                                           
  | 페이지 | 설명 |                                                                                                        
  |---|---|                                                                                                              
  | `home.html` | 포트폴리오 메인 — 6개 섹션 (Intro / About / Skills / Projects / Career / Devlog) |
  | `devlogHome.html` | 개발 블로그 홈 — 게시글 목록 · 인기글 · 최근 댓글 · 태그 영역 |                                    
  | `playground.html` | 실험·실습 공간 |                                                                                   
                                                                                                                           
  ## 주요 구현 기능                                                                                                     
                                                                                                                         
  라이브러리 없이 DOM API 직접 활용:                                                                                       
                                                                                                                         
  - **HTML5 Video API**: 배경 영상 재생 / 일시정지 직접 제어                                                               
  - **자체 캐러셀**: 좌우 화살표 · 점 네비게이션 (Slick 등 라이브러리 미사용)
  - **카드 플립 효과**: CSS Transform + JS 클래스 토글                                                                     
  - **스킬 카드 전환**: 동적 콘텐츠 교체                                                                                 
  - **반응형 디자인**: 미디어쿼리 + 모바일 메뉴 분기                                                                       
  - **스크롤 인터랙션**: 탑 버튼, 부드러운 섹션 이동                                                                       
                                                                                                                           
  `home.js` 약 36KB 규모, 위 인터랙션을 모듈 함수 단위로 분리.                                                             
                                                                                                                         
  ## 🌐 배포 환경                                                                                                          
                                                                                                                         
  - **호스팅**: AWS
  - **현재 상태**: AWS 요금 관계로 배포 중단 — 코드 검토용
